# task2: Wasdom ESP32 operation algorithm

**1. Download  Arduino IDE**

**2. In your Arduino IDE, go to File> Preferences**

<img width="554" alt="image" src="https://user-images.githubusercontent.com/108360083/182369586-5a7251a8-08f1-47c7-b14d-7e2cd5a3b025.png">


**3. Enter the following into the “Additional Board Manager URLs” field:**

https://dl.espressif.com/dl/package_esp32_index.json

<img width="456" alt="image" src="https://user-images.githubusercontent.com/108360083/182368256-ba2c0411-c7a1-45dd-b448-3f85e30819cf.png">

**Then, click the “OK” button:**

**4. Open the Boards Manager. Go to Tools > Board > Boards Manager**

<img width="558" alt="image" src="https://user-images.githubusercontent.com/108360083/182369677-d9d59d9e-7b51-4289-8ee0-e4c80c8c2087.png">

**5. Search for ESP32 and press install button for the “ESP32 by Espressif Systems**

<img width="588" alt="image" src="https://user-images.githubusercontent.com/108360083/182370781-7c2052b7-b93d-4be4-a51b-f62a6e434072.png">

**6. Select your Board in Tools > Board menu > ESP32 Arduino > WEMOS D1 MINI ESP32**


<img width="799" alt="image" src="https://user-images.githubusercontent.com/108360083/182373981-19a07db0-4a92-43a3-80f2-87849a2a79d8.png">

**7. Select the Port**

<img width="368" alt="image" src="https://user-images.githubusercontent.com/108360083/182376786-9c438fa0-43e4-4574-8eac-1dcbf6bdb48b.png">

**8. Open the following example under File > Examples > Basics > Blink**

<img width="683" alt="image" src="https://user-images.githubusercontent.com/108360083/182377316-ae20fe60-7ed6-4f46-989c-159da439eb93.png">

**9. A new sketch opens in your Arduino IDE**
 
 <img width="886" alt="image" src="https://user-images.githubusercontent.com/108360083/182378187-e600ac36-151d-4edb-b0cc-1b2857c2467f.png">
